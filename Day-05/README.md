# 🗓️ Day 05: CSS Grid Architecture (Community Initiatives Gallery)

## 🎯 Objective

Today's goal was to learn **CSS Grid** and build a clean, responsive two-dimensional layout for showcasing TechNova Community initiatives.

Unlike Flexbox, which is mainly used for one-dimensional layouts, CSS Grid allows developers to create structured rows and columns with minimal code.

---

## 🚀 What I Built

A fully responsive **Community Initiatives Gallery** featuring:

- 🚀 Hackathon Hub
- 💻 Open Source Projects
- 🤖 AI & ML Workshops
- 🌐 Web Development Bootcamp
- ☁️ Cloud Computing

The section is built using **CSS Grid** with equal-width columns and modern card styling.
